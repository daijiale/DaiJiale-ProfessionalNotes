Shadow Boxes

These boxes can created just only applying class .box-shadow-1 to .box-shadow-5

Note: To use box shadow with funny boxes you need to wrap funny box in a wrapper div then apply .box-shadow-* classes to that parent div 

<div class="box-shadow-*">
	<div class="funny-box">
		...
	</div>
</div>
